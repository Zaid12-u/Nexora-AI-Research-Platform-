# Nexora — AI Research Platform

> Discover Research Smarter with AI

---

## Features

- Semantic Search — AI understands meaning, not just keywords
- Gap Finder — Discover what's missing in research
- Deep Search Mode — AI summaries of top 5 papers
- Search History — Track and revisit your research
- Live Fallback — Fetches from OpenAlex if topic not in DB

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React + Vite + Tailwind CSS |
| Auth Backend | Node.js + Express |
| AI Backend | FastAPI + Python |
| AI Model | Sentence Transformers `all-MiniLM-L6-v2` |
| Database | MongoDB Atlas |
| Vector Search | MongoDB Atlas Vector Search |
| Paper Source | OpenAlex API |
